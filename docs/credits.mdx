# Credits & Billing

API access uses a credit-based system. Each API call deducts a small amount from your credit balance. Some endpoints are completely free.

## Pricing Table

| Endpoint | Cost per Call | Scope |
|---|---|---|
| `GET /ideas` | $0.01 | `ideas:read` |
| `GET /ideas/search` | $0.02 | `ideas:read` |
| `GET /ideas/:slug` | $0.01 | `ideas:read` |
| `GET /categories` | **Free** | `categories:read` |
| `GET /account/credits` | **Free** | `account:read` |

## Free Credit

<Tip>
Every new developer gets **$5.00 free credit** when generating their first API key. No credit card required — just sign up and start building.
</Tip>

That's enough for:
- **500** list or detail requests (`GET /ideas`, `GET /ideas/:slug`)
- **250** search requests (`GET /ideas/search`)
- Unlimited `GET /categories` and `GET /account/credits` calls

## Adding Credits

Top up your balance from the [Developer Dashboard](https://opensaasidea.com/developer/api) with any custom amount between **$5 and $500**. Payments are processed securely via Dodo Payments.

<Steps>
  <Step title="Go to Developer Dashboard">
    Visit [opensaasidea.com/developer/api](https://opensaasidea.com/developer/api)
  </Step>
  <Step title="Enter amount">
    Choose a preset ($10, $25, $50) or enter a custom amount.
  </Step>
  <Step title="Complete payment">
    Pay securely via Dodo Payments. Credits are added to your balance instantly.
  </Step>
</Steps>

## Checking Your Balance

Use the free `/account/credits` endpoint to check your balance programmatically:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://qpfmnheghocfcckferum.supabase.co/functions/v1/public-api/account/credits
```

```json Response
{
  "balance": 4.50,
  "balance_cents": 450,
  "lifetime_purchased": 10.00,
  "lifetime_used": 0.50,
  "currency": "USD"
}
```

## Insufficient Credits

If your balance is too low for a request, you'll receive a `402 Payment Required` response:

```json 402 — Insufficient Credits
{
  "error": "insufficient_credits",
  "message": "Not enough API credits. Current balance: $0.03",
  "balance_cents": 3,
  "cost_cents": 2
}
```

<Warning>
Credits are non-refundable. Monitor your balance using the `/account/credits` endpoint to avoid interruptions.
</Warning>
