# List Categories

Returns all available idea categories with the number of ideas in each. This is a **free endpoint** — no credits are charged.

**Cost:** Free | **Scope:** `categories:read`

## Response

<ResponseField name="categories" type="array" required>
  Array of category objects.
  <Expandable title="Category object">
    <ResponseField name="name" type="string">Category name.</ResponseField>
    <ResponseField name="count" type="integer">Number of ideas in this category.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="total" type="integer" required>
  Total number of categories.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://opensaasidea.com/v1/api/categories"
```

```javascript JavaScript
const res = await fetch(
  "https://opensaasidea.com/v1/api/categories",
  { headers: { Authorization: "Bearer YOUR_API_KEY" } }
);
const data = await res.json();
```

```python Python
import requests

res = requests.get(
    "https://opensaasidea.com/v1/api/categories",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)
data = res.json()
```
