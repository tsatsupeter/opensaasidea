---
title: "List Categories"
api: "GET /categories"
description: "Get all available idea categories with their idea counts."
---

Returns all available idea categories sorted by idea count (descending). This is a **free endpoint**.

**Cost:** Free | **Scope:** `categories:read`

## Response

<ResponseField name="data" type="array" required>
  Array of category objects.
  <Expandable>
    <ResponseField name="name" type="string">Category name.</ResponseField>
    <ResponseField name="count" type="integer">Number of public ideas in this category.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object" required>
  <Expandable>
    <ResponseField name="total" type="integer">Total number of categories.</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://openprojectidea.com/v1/api/categories"
```

```javascript JavaScript
const res = await fetch(
  "https://openprojectidea.com/v1/api/categories",
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { data } = await res.json();
```

```python Python
import requests

res = requests.get(
    "https://openprojectidea.com/v1/api/categories",
    headers={"x-api-key": "YOUR_API_KEY"}
)
categories = res.json()["data"]
```
</RequestExample>
