---
title: "Platform Stats"
api: "GET /stats"
description: "Get platform-wide statistics including totals and top categories."
---

## Response

<ResponseField name="data" type="object">
  <Expandable>
    <ResponseField name="total_ideas" type="integer">
      Total number of public ideas on the platform.
    </ResponseField>
    <ResponseField name="total_votes" type="integer">
      Total votes cast across all ideas.
    </ResponseField>
    <ResponseField name="total_comments" type="integer">
      Total comments posted.
    </ResponseField>
    <ResponseField name="total_users" type="integer">
      Total registered users.
    </ResponseField>
    <ResponseField name="top_categories" type="array">
      Top 10 categories by idea count.
    </ResponseField>
  </Expandable>
</ResponseField>

## Scope

`stats:read` - Free

<RequestExample>
```bash cURL
curl -H "x-api-key: osk_your_key" \
  https://openprojectidea.com/v1/api/stats
```

```javascript JavaScript
const res = await fetch(
  "https://openprojectidea.com/v1/api/stats",
  { headers: { "x-api-key": "osk_your_key" } }
);
const { data } = await res.json();
console.log(`${data.total_ideas} ideas, ${data.total_users} users`);
```
</RequestExample>
