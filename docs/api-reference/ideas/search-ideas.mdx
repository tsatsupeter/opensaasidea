# Search Ideas

Search public ideas by title, tagline, or category. Returns matching results with pagination.

**Cost:** $0.02 per call | **Scope:** `ideas:read`

## Query Parameters

<ParamField query="q" type="string" required>
  Search query. Searches across title, tagline, and category fields.
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Number of results per page (1–50).
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Pagination offset.
</ParamField>

## Response

<ResponseField name="ideas" type="array" required>
  Array of matching idea objects (same schema as List Ideas).
</ResponseField>

<ResponseField name="total" type="integer" required>
  Total number of matching ideas.
</ResponseField>

<ResponseField name="limit" type="integer" required>
  Number of results returned.
</ResponseField>

<ResponseField name="offset" type="integer" required>
  Current pagination offset.
</ResponseField>

<ResponseField name="query" type="string" required>
  The search query that was used.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://opensaasidea.com/v1/api/ideas/search?q=ai%20resume&limit=10"
```

```javascript JavaScript
const res = await fetch(
  "https://opensaasidea.com/v1/api/ideas/search?q=ai+resume&limit=10",
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const data = await res.json();
```

```python Python
import requests

res = requests.get(
    "https://opensaasidea.com/v1/api/ideas/search",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"q": "ai resume", "limit": 10}
)
```
