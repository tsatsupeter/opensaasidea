---
title: "Search Ideas"
api: "GET /ideas/search"
description: "Full-text search across titles, taglines, descriptions, and categories."
---

Search public ideas by keyword. Searches across title, tagline, description, and category fields. Results are sorted by relevance (vote score). Response fields vary by [API mode](/modes).

**Cost:** $0.02 per call | **Scope:** `ideas:read`

## Query Parameters

<ParamField query="q" type="string" required>
  Search query string.
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Number of results per page (max 50).
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Pagination offset.
</ParamField>

## Response

<ResponseField name="data" type="array" required>
  Array of matching idea objects (same fields as [List Ideas](/api-reference/ideas/list-ideas)).
</ResponseField>

<ResponseField name="meta" type="object" required>
  <Expandable>
    <ResponseField name="total" type="integer" />
    <ResponseField name="limit" type="integer" />
    <ResponseField name="offset" type="integer" />
    <ResponseField name="query" type="string">The search query used.</ResponseField>
    <ResponseField name="mode" type="string">`broad` or `lite`</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://openprojectidea.com/v1/api/ideas/search?q=ai%20resume&limit=10"
```

```javascript JavaScript
const res = await fetch(
  "https://openprojectidea.com/v1/api/ideas/search?q=ai+resume&limit=10",
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { data, meta } = await res.json();
```

```python Python
import requests

res = requests.get(
    "https://openprojectidea.com/v1/api/ideas/search",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"q": "ai resume", "limit": 10}
)
```
</RequestExample>
