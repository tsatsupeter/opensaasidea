---
title: "Get Trending Ideas"
api: "GET /ideas/trending"
description: "Get the top trending ideas sorted by votes and views."
---

## Query Parameters

<ParamField query="limit" type="integer" default="10">
  Number of ideas to return (max 25).
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of trending idea objects.
</ResponseField>

<ResponseField name="meta" type="object">
  <Expandable>
    <ResponseField name="limit" type="integer" />
    <ResponseField name="mode" type="string">
      `broad` or `lite`
    </ResponseField>
  </Expandable>
</ResponseField>

## Scope

`ideas:read` - $0.01 per call

<RequestExample>
```bash cURL
curl -H "x-api-key: osk_your_key" \
  https://openprojectidea.com/v1/api/ideas/trending?limit=5
```

```javascript JavaScript
const res = await fetch(
  "https://openprojectidea.com/v1/api/ideas/trending?limit=5",
  { headers: { "x-api-key": "osk_your_key" } }
);
const { data } = await res.json();
```
</RequestExample>
