# Get Idea

Retrieve full details for a single idea by its slug or UUID. Includes tech stack, pricing tiers, and detailed metrics.

**Cost:** $0.01 per call | **Scope:** `ideas:read`

## Path Parameters

<ParamField path="slug" type="string" required>
  The idea's URL slug (e.g. `ai-resume-builder`) or UUID.
</ParamField>

## Response

<ResponseField name="idea" type="object" required>
  Full idea object.
  <Expandable title="Idea object">
    <ResponseField name="id" type="string">UUID of the idea.</ResponseField>
    <ResponseField name="title" type="string">Title of the idea.</ResponseField>
    <ResponseField name="slug" type="string">URL-friendly slug.</ResponseField>
    <ResponseField name="tagline" type="string">Short tagline.</ResponseField>
    <ResponseField name="description" type="string">Full description of the idea.</ResponseField>
    <ResponseField name="category" type="string">Category name.</ResponseField>
    <ResponseField name="difficulty" type="string">Difficulty level (Easy, Medium, Hard).</ResponseField>
    <ResponseField name="estimated_mrr_low" type="integer">Low MRR estimate in USD.</ResponseField>
    <ResponseField name="estimated_mrr_high" type="integer">High MRR estimate in USD.</ResponseField>
    <ResponseField name="estimated_users" type="string">Estimated user base range.</ResponseField>
    <ResponseField name="time_to_build" type="string">Estimated build time.</ResponseField>
    <ResponseField name="platforms" type="array">Target platforms (Web, Mobile, Desktop, etc.).</ResponseField>
    <ResponseField name="tech_stack" type="object">Recommended technology stack.</ResponseField>
    <ResponseField name="pricing_tiers" type="array">Suggested pricing tiers for the idea.</ResponseField>
    <ResponseField name="upvotes" type="integer">Number of upvotes.</ResponseField>
    <ResponseField name="downvotes" type="integer">Number of downvotes.</ResponseField>
    <ResponseField name="view_count" type="integer">Total view count.</ResponseField>
    <ResponseField name="created_at" type="string">ISO 8601 timestamp.</ResponseField>
  </Expandable>
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://opensaasidea.com/v1/api/ideas/ai-resume-builder"
```

```javascript JavaScript
const res = await fetch(
  "https://opensaasidea.com/v1/api/ideas/ai-resume-builder",
  { headers: { Authorization: "Bearer YOUR_API_KEY" } }
);
const idea = (await res.json()).idea;
```

```python Python
import requests

res = requests.get(
    "https://opensaasidea.com/v1/api/ideas/ai-resume-builder",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)
idea = res.json()["idea"]
```
