---
title: "Get Idea"
api: "GET /ideas/{slug}"
description: "Retrieve full details for a single idea by slug or UUID."
---

Get complete details for a single idea. The depth of data returned depends on [API mode](/modes) (broad vs lite). Broad mode includes revenue breakdowns, marketing strategies, team roles, competitors, and more.

**Cost:** $0.01 per call | **Scope:** `ideas:read`

## Path Parameters

<ParamField path="slug" type="string" required>
  The idea's URL slug (e.g. `ai-resume-builder`) or UUID.
</ParamField>

<RequestExample>
```bash cURL
# Broad (full detail)
curl -H "x-api-key: YOUR_API_KEY" \
  "https://openprojectidea.com/v1/api/ideas/ai-resume-builder"

# Lite (SaaS-focused)
curl -H "x-api-key: YOUR_API_KEY" \
  "https://opensaasidea.com/v1/api/ideas/ai-resume-builder"
```

```javascript JavaScript
const res = await fetch(
  "https://openprojectidea.com/v1/api/ideas/ai-resume-builder",
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { data } = await res.json();
```

```python Python
import requests

res = requests.get(
    "https://openprojectidea.com/v1/api/ideas/ai-resume-builder",
    headers={"x-api-key": "YOUR_API_KEY"}
)
idea = res.json()["data"]
```
</RequestExample>
