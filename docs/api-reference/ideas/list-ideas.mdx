---
title: "List Ideas"
api: "GET /ideas"
description: "Retrieve a paginated list of public ideas with filtering and sorting."
---

Returns a paginated list of public ideas. Supports filtering by category/platform and sorting. Response fields vary by [API mode](/modes) (broad vs lite).

**Cost:** $0.01 per call | **Scope:** `ideas:read`

## Query Parameters

<ParamField query="limit" type="integer" default="20">
  Number of results per page (max 50).
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Pagination offset.
</ParamField>

<ParamField query="category" type="string">
  Filter by category name (e.g. `Developer Tools`, `FinTech`).
</ParamField>

<ParamField query="platform" type="string">
  Filter by platform (e.g. `web`, `mobile`, `desktop`).
</ParamField>

<ParamField query="sort" type="string" default="newest">
  Sort order: `newest`, `popular`, `mrr`, or `views`.
</ParamField>

## Response

<ResponseField name="data" type="array" required>
  Array of idea objects. Fields depend on broad vs lite mode.
  <Expandable title="Idea object (broad)">
    <ResponseField name="id" type="string" />
    <ResponseField name="title" type="string" />
    <ResponseField name="slug" type="string" />
    <ResponseField name="tagline" type="string" />
    <ResponseField name="category" type="string" />
    <ResponseField name="platform" type="string" />
    <ResponseField name="monetization_model" type="string" />
    <ResponseField name="estimated_mrr_low" type="integer" />
    <ResponseField name="estimated_mrr_high" type="integer" />
    <ResponseField name="vote_score" type="integer" />
    <ResponseField name="upvotes" type="integer">Broad only</ResponseField>
    <ResponseField name="downvotes" type="integer">Broad only</ResponseField>
    <ResponseField name="views" type="integer">Broad only</ResponseField>
    <ResponseField name="comment_count" type="integer">Broad only</ResponseField>
    <ResponseField name="created_at" type="string" />
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object" required>
  <Expandable>
    <ResponseField name="total" type="integer" />
    <ResponseField name="limit" type="integer" />
    <ResponseField name="offset" type="integer" />
    <ResponseField name="mode" type="string">`broad` or `lite`</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
# Broad API
curl -H "x-api-key: YOUR_API_KEY" \
  "https://openprojectidea.com/v1/api/ideas?limit=5&sort=popular"

# Lite API
curl -H "x-api-key: YOUR_API_KEY" \
  "https://opensaasidea.com/v1/api/ideas?limit=5&sort=popular"
```

```javascript JavaScript
const res = await fetch(
  "https://openprojectidea.com/v1/api/ideas?limit=5&sort=popular",
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { data, meta } = await res.json();
```

```python Python
import requests

res = requests.get(
    "https://openprojectidea.com/v1/api/ideas",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"limit": 5, "sort": "popular"}
)
result = res.json()
ideas = result["data"]
```
