# Get Credits

Check your current API credit balance, lifetime purchases, and lifetime usage. This is a **free endpoint** — no credits are charged.

**Cost:** Free | **Scope:** `account:read`

## Response

<ResponseField name="balance" type="number" required>
  Current balance in dollars (e.g. `4.50`).
</ResponseField>

<ResponseField name="balance_cents" type="integer" required>
  Current balance in cents (e.g. `450`).
</ResponseField>

<ResponseField name="lifetime_purchased" type="number" required>
  Total amount purchased in dollars.
</ResponseField>

<ResponseField name="lifetime_used" type="number" required>
  Total amount used in dollars.
</ResponseField>

<ResponseField name="currency" type="string" required>
  Always `"USD"`.
</ResponseField>

## Example

<CodeGroup>

```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://opensaasidea.com/v1/api/account/credits"
```

```javascript JavaScript
const res = await fetch(
  "https://opensaasidea.com/v1/api/account/credits",
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const credits = await res.json();
console.log(`Balance: $${credits.balance}`);
```

```python Python
import requests

res = requests.get(
    "https://opensaasidea.com/v1/api/account/credits",
    headers={"x-api-key": "YOUR_API_KEY"}
)
credits = res.json()
print(f"Balance: ${credits['balance']}")
```

</CodeGroup>
