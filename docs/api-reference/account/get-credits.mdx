---
title: "Get Credits"
api: "GET /account/credits"
description: "Check your current API credit balance and usage."
---

Check your current API credit balance, lifetime purchases, and lifetime usage. This is a **free endpoint**.

**Cost:** Free | **Scope:** `account:read`

## Response

<ResponseField name="data" type="object" required>
  <Expandable>
    <ResponseField name="balance" type="number">Current balance in dollars (e.g. `4.50`).</ResponseField>
    <ResponseField name="balance_cents" type="integer">Current balance in cents (e.g. `450`).</ResponseField>
    <ResponseField name="lifetime_purchased" type="number">Total amount purchased in dollars.</ResponseField>
    <ResponseField name="lifetime_used" type="number">Total amount used in dollars.</ResponseField>
    <ResponseField name="currency" type="string">Always `"USD"`.</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -H "x-api-key: YOUR_API_KEY" \
  "https://openprojectidea.com/v1/api/account/credits"
```

```javascript JavaScript
const res = await fetch(
  "https://openprojectidea.com/v1/api/account/credits",
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const { data } = await res.json();
console.log(`Balance: $${data.balance}`);
```

```python Python
import requests

res = requests.get(
    "https://openprojectidea.com/v1/api/account/credits",
    headers={"x-api-key": "YOUR_API_KEY"}
)
credits = res.json()["data"]
print(f"Balance: ${credits['balance']}")
```
</RequestExample>
