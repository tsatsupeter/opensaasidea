## Notes
- Prefer `.svg` format for crisp rendering at any size
- If only `.png` is available, that's fine — just use the same filename with `.png`
- Keep logos under 50KB each for performance
- After adding logos, update `/src/lib/affiliates.ts` with a `logo` field
